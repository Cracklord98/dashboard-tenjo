# Backend - Plan de Desarrollo Tenjo

Coloca aquí tu archivo Excel `PLAN INDICATIVO TENJO.xlsx`.

El archivo debe contener las siguientes columnas principales (el parser es flexible con nombres):

## Columnas Obligatorias:
- **EJE** - Eje estratégico del Plan de Desarrollo
- **PROGRAMA PDT** - Nombre del programa
- **SUBPROGRAMA** - Nombre del subprograma
- **META DE RESULTADO** / **MR** - Meta de resultado
- **INDICADOR DE RESULTADO** - Indicador de la meta
- **NOMBRE DEL PROYECTO** - Nombre del proyecto
- **META DE PRODUCTO** - Meta de producto específica
- **INDICADOR** - Indicador de producto

## Columnas de Avance Físico (2025):
- **T1 PLANEADO 2025** / **T1 EJECUTADO 2025**
- **T2 PLANEADO 2025** / **T2 EJECUTADO 2025**
- **T3 PLANEADO 2025** / **T3 EJECUTADO 2025**
- **T4 PLANEADO 2025** / **T4 EJECUTADO 2025**
- **TOTAL PLANEADO 2025** / **TOTAL EJECUTADO 2025**
- **AVANCE 2025** - Porcentaje de avance

## Columnas Financieras (2025):
- **APROPIACION DEFINITIVA 2025** - Presupuesto aprobado
- **COMPROMISOS 2025** - Compromisos adquiridos
- **PAGOS 2025** - Pagos realizados
- **% EJECUCIÓN FINANCIERA** - Porcentaje de ejecución

## Columnas Adicionales:
- **PONDERADOR EJE** - Peso del eje en el plan
- **BPIN** - Código del proyecto
- **RESPONSABLE** - Dependencia responsable
- **UNIDAD DE MEDIDA** - Unidad de medición
- **L.B** - Línea base
- **VALOR ESPERADO CUATRIENIO** - Meta 2024-2027
- **ESTADO** - Estado de la meta
- **SOPORTES DE CUMPLIMIENTO** - Evidencias
